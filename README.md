# Tetromino Stacking

Desktop app for dragging, rotating, grouping, and saving tetromino layouts.

## Requirements

- Python 3.10 or newer
- Tkinter support (included by default on most Windows/macOS Python installs)

### Install Tkinter if needed (Linux)

- Debian/Ubuntu/Mint:

~~~bash
sudo apt update
sudo apt install python3-tk
~~~

- Fedora:

~~~bash
sudo dnf install python3-tkinter
~~~

- Arch/Manjaro:

~~~bash
sudo pacman -S tk
~~~

- openSUSE:

~~~bash
sudo zypper install python3-tk
~~~
## Run

~~~bash
python app.py
~~~
